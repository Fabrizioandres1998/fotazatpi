# Fotaza 2

Proyecto final de Programación Web II.

Aplicación web para compartir fotografías en línea desarrollada con Node.js, Express, Pug, Sequelize y MySQL.

---

## Tecnologías utilizadas

- Node.js
- Express
- Pug
- Sequelize
- MySQL
- Bootstrap
- bcrypt
- express-session

---

## Instalación local

1. Clonar repositorio

```bash
git clone URL_DEL_REPO
```

2. Instalar dependencias

```bash
npm install
```

3. Configurar variables de entorno

Crear archivo `.env`

```env
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
SESSION_SECRET=
```

4. Ejecutar migraciones

```bash
npx sequelize db:migrate
```

5. Iniciar aplicación

```bash
npm start
```
